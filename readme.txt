-[ Основные изменения ]-------------------------------------------------------

  1. Исправленные ошибки:

1.1. Шаман лечит при положительном ответе на соответствующей вопрос.
1.2. При старте новой игры очищается флаг начала войны из-за отказа жениться.
1.3. При прибытии каравана очищается соответствующий флаг. Это позволяет
     отправлять караван несколько раз за игру.


  2. Изменения.

2.1. Убраны просьбы королевы и вопросы про новогодний бал.
2.2. Норма зерна 70%.
2.3. Сразу автоматически распределяется зерно, больше не задается вопрос о том
     хотите ли распределить вручную.
2.4. При смерти королевы, не задается вопрос о том, хотите ли впустить гонца
     от нее.

Примечание: можно отключить все эти изменения, если запустить игру с
            параметром командной строки "vanilla" (без кавычек). Например:
            "EIFORIA.EXE vanilla" (без кавычек).


  3. Новые возможности.

3.1. Игру можно сохранять.
     Примечание 1: На данный момент есть только автоматическое сохранение
                   (перед концом каждого хода). РУЧНОГО СОХРАНЕНИЯ НЕТ.
     Примечание 2: СУЩЕСТВУЮЩИЕ ФАЙЛЫ ПЕРЕЗАПИСЫВАЮТСЯ БЕЗО ВСЯКИХ
                   ДОПОЛНИТЕЛЬНЫХ ВОПРОСОВ.
2.2. Игру можно загружать, для этого надо при запуске игры указать в командной
     строке имя файла сохранения, например "EIFORIA.EXE SAVE\27.SAV" (без
     кавычек). Можно также загрузить последний файл автосохранения, для этого
     надо запустить игру с параметром "loadlast" (без кавычек), например:
     "EIFORIA.EXE loadlast" (без кавычек).
3.3. В комплетке поставляется также исполняемый файл для Windows
     (eif-win.exe).


  4. Известные проблемы.

4.1. При автосохранении не учитываются деньги, потраченные на новогодний бал.


Подробнее об мзменениях можно прочитать в файле HISTORY.TXT




-[ Командная строка ]---------------------------------------------------------

  1. При запуске игры распознаются следующие параметры командной строки:

  no|nointro              Отключение заставки.
  vanilla|orig|original   Режим игры как в оригинале:
                          - вопросы про новогодний бал;
                          - просьбы королевы;
                          - норма зерна 100%;
                          - каждый ход спрашивается, хотите ли сами
                            распределить зерно.
  noautosave              Отключение автосохранения.
  [имя файла сохранения]  Загрузить сохранённую игру (имя файла обязано иметь
                          расширение).
  [loadlast]              Загрузить последний файл автосохранения.
  new|newgame             Сразу начать новую игру, не показывая главное меню.


  2. Примечания.

2.1. Параметры командной строки являются регистронезавиcимыми.
2.2. Параметры командной строки, кроме имени файла сохранённия, могут
     начинаться с префикса "/" (без кавычек) или "-" (без кавычек).


  3. Примеры использования.

3.1. Отключить вывод заставки и включить режим игры как в оригинале:

     --------------------------------
     EIFORIA.EXE no vanilla
     --------------------------------

     или:

     --------------------------------
     EIFORIA.EXE -nointro -orig
     --------------------------------

     или:

     --------------------------------
     EIFORIA.EXE /nointro /original
     --------------------------------

3.2. Загрузить сохранённую игру:

     --------------------------------
     EIFORIA.EXE SAVE\2.SAV
     --------------------------------

3.3. Загрузить последнюю автосохранённую игру:

     ------------------------
     EIFORIA.EXE loadlast
     ------------------------

     или:

     ------------------------
     EIFORIA.EXE -loadlast
     ------------------------

     При загрузке последней автосохраненной игры, программа ищет во всех
     подкаталогах из каталога SAVE самый новый файл автосохранения и загружает
     его.
